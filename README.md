## 🔍 About

**Flower** is a creative coding mini-project where I explored the fundamentals of **HTML5 Canvas / CSS animations** to draw and animate beautiful flower patterns programmatically.

This project is part of my journey to get comfortable with front-end technologies and creative visual programming. It's a great exercise for understanding:
- CSS keyframe animations
- Mathematical patterns in code (petals, symmetry)
- DOM manipulation with JavaScript

## ⚙️ Installation

```bash
# Clone the repository
git clone https://github.com/VINAYBOYAT11/flower.git
cd flower

# Open in browser
# Simply open index.html in any browser
start index.html    # Windows
open index.html     # macOS
```

No build tools or dependencies required! ✅
